## Statement
本项目现阶段旨在尽量模拟各种场景来测试[MusicLibrary](https://github.com/lizixian18/MusicLibrary)，并作为一个例子去使用[MusicLibrary](https://github.com/lizixian18/MusicLibrary)，
所以做得比较粗糙，界面也不多，以后会慢慢完善起来，暂不提供Apk下载，有兴趣可以下载代码运行一下，一起学习。
